Check Sudoku
============

Este es un ejercicio de clase que consiste en crear un programa 
que compruebe si un sudoku cumple con las condiciones para ser válido.

Para ello es necesario comprobar que se cumplan cuatro condiciones:
1. El sudoku es cuadrado, es decir, tiene el mismo número de filas que de columnas
2. Sus valores son números enteros positivos en un rango de entre 1 y la longitud de cada fila
3. Los números no pueden repetirse en una misma fila
4. Los números no pueden repetirse en una misma columna
